  1. 本地windows安装docker-compose运行环境
  2. 默认端口需求
  3. JDK14允许环境

运行

  • build_image.cmd 打包项目为docker image
  • docker-compose.yml docker运行配置
# 1.打开build_image.cmd 修改JAVA_HOME 环境变量
set JAVA_HOME=D:\Program Files\Java\jdk-14.0.1

# 2.运行 build_image.cmd
build_image.cmd

# 3.启动项目
docker-compose up -d
